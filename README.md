## СУТЬ ПРОЕКТА
Представляет из себя сайт с аналитикой и поиском игр на игровых платформах, в основном Steam. Используется сторонняя API CheapShark.
## ВОЗМОЖНОСТИ
На моем сайте можно увидеть все существующие скидки в стиме на данные момент, а также просмотреть аналитику конкретной игры(самая низкая цена за все время, дата когда она была, а также платформы где продается игра и цены). Поиск игр из Steam по названию.
